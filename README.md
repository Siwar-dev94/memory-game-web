# Memory Game Web Application

# Description
This project is a web-based memory game where players must remember and match cards based on displayed names within a limited time.
# Features
16-card grid (2 × 8 layout)
Countdown timer (60 seconds)
Random card shuffling
Card hiding after memorization phase
Random name selection
Score calculation and game over logic
# Technologies
HTML
CSS
JavaScript
# How to Play
Memorize the cards within 60 seconds
Cards are hidden automatically
Click the correct card based on the displayed name
Game ends on wrong answer
